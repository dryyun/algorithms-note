# 啊哈!算法 - 第二章

严格来说，这章是讲数据结构，而不是算法。    
不过数据结构和算法，相辅相成，都要掌握。  

## 队列 Queue

FIFO，先进先出数据结构

在头部 head 进行删除操作，出队  
在尾部 tail 进行增加操作，入队  
当 head = tail 时为空队列  

队列的实现，应该来说是通过 list 链表，不过这里我按照书中的例子，使用了数组实现  

## 栈 Stack

先进后出数据结构  

只能在一端进行插入和删除操作  

一维数组 + top 指向顶部实现了`栈` ，top = 0 为空栈

栈的应用 
- 判断回文字符串  
- 验证括号匹配  


## 链表 List

链表是一种在某些场景下比数组方便的数据结构  
list 有很多类型 
- 单向链表
- 双向链表
- 循环链表

书中目前就介绍了最简单的`单向链表`
