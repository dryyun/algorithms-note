# 颠倒二进制位

## 链接
https://leetcode-cn.com/problems/reverse-bits/description/

## 描述
颠倒给定的 32 位无符号整数的二进制位。  

示例:
```text
输入: 43261596
输出: 964176192
解释: 43261596 的二进制表示形式为 00000010100101000001111010011100 ，
     返回 964176192，其二进制表示形式为 00111001011110000010100101000000 。
```
