# Excel表列名称

## 链接
https://leetcode-cn.com/problems/excel-sheet-column-title/description/

## 描述
给定一个正整数，返回它在 Excel 表中相对应的列名称。  

例如，
```text
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
```

示例 1:
```text
输入: 1
输出: "A"
````
示例 2:
```text
输入: 28
输出: "AB"
```
示例 3:
```text
输入: 701
输出: "ZY"
```
