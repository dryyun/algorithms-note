# 最长有效括号

## 链接
https://leetcode-cn.com/problems/longest-valid-parentheses/description/

## 描述
给定一个只包含 '(' 和 ')' 的字符串，找出最长的包含有效括号的子串的长度。

示例 1:
```text
输入: "(()"
输出: 2
解释: 最长有效括号子串为 "()"
```

示例 2:
```text
输入: ")()())"
输出: 4
解释: 最长有效括号子串为 "()()"
```


## 思路
先各自匹配括号，求出所有合适的起始 index ，然后归并 index ，能否合起来

比如 `)())(())` ，[ 1, 2 ], [ 5, 6 ], [ 4, 7 ] ，=> [4,7] => 4 
比如 `)()(()())`， [ 1, 2 ], [ 4, 5 ], [ 6, 7 ], [ 3, 8 ] =》 [1,8] => 8
