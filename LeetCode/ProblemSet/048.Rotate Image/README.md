# 旋转图像

## 链接
https://leetcode-cn.com/problems/rotate-image/description/

## 描述
给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

说明：
你必须在原地旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

示例 1:
```text
给定 matrix = 
[
  [1,2,3],
  [4,5,6],
  [7,8,9]
],

原地旋转输入矩阵，使其变为:
[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]
```

示例 2:
```text
给定 matrix =
[
  [ 5, 1, 9,11],
  [ 2, 4, 8,10],
  [13, 3, 6, 7],
  [15,14,12,16]
], 

原地旋转输入矩阵，使其变为:
[
  [15,13, 2, 5],
  [14, 3, 4, 1],
  [12, 6, 8, 9],
  [16, 7,10,11]
]
``` 

## 思路
```text
以
[
  [5, 1, 9, 11],
  [2, 4, 8, 10],
  [13, 3, 6, 7],
  [15, 14, 12, 16]
];
为例
```

```text

[0][0] = 5 => [0][3] = 5
[0][1] = 1 => [1][3] = 1
[0][2] = 9 => [2][3] = 9 
[0][3] = 11 => [3][3] = 11

[1][0] = 2 => [0][2] = 2
[1][1] = 4 => [1][2] = 4
[1][2] = 8 => [2][2] = 8
[1][3] = 10 => [3][2] = 10

同一行的变成了同一列
旋转结果 x = oy, y = len - 1 - ox 
```
