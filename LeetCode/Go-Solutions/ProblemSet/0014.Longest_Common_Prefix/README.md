# 14. 最长公共前缀

## 链接
https://leetcode-cn.com/problems/longest-common-prefix/

## 难度
简单

## 描述

编写一个函数来查找字符串数组中的最长公共前缀。  

如果不存在公共前缀，返回空字符串 ""。  

示例 1:
```text
输入: ["flower","flow","flight"]
输出: "fl"
```
示例 2:
```text
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```
说明:  

所有输入只包含小写字母 a-z 。 

## 思路
水平扫描，
- 第一个字符串的第一个字节和其余其余字符串的第一个字节比较
- 第一个字符串的第二个字节和其余其余字符串的第二个字节比较

使用 sort.Strings 是按照字符串的字节逐个比较
