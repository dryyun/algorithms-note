## 内存的工作原理

将数据存储到内存时，请求计算机提供存储空间，计算机给你一个存储地址。

## 数组和链表

数组
- 优点，读取元素方便，O(1)
- 缺点，插入元素，必须整个数组连续插入，如果没有连续内容，要重新分配，O(n) 
链表
- 优点，插入元素方便，可存储在任何地方，只要把地址告诉上一个元素就行，O(1) ，同样的删除元素也方便  
- 缺点，跳跃读取某个元素不方便，只能从头开始读，O(n)  

### 不同的使用场景

1、在中间插入元素 - 链表
2、删除某个元素 - 链表
3、插入操作多，读取操作少 - 链表
